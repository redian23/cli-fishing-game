# Title:  console-fishing

## Bash script to play fishing on Terminal

## To start localy script:
```
./fishing
or 
bash fishing
```
## Positional Parameters
Enable Demo mode:
``
(-dm)(--demo)
``

Enable Debug mode:
``
(-d)(--debug)
``

## Install 
Script will be copy in /usr/local/bin direcroty.
Need root password to install/update/remove.  
```
./install.sh
```

## Run
user@domain:$ ``fishing `` 

## Update
```
./update.sh
```

## Remove
```
./remove.sh
```
